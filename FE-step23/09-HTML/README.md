## Why?

웹은 콘텐츠를 제공한다.

콘텐츠의 구성은 웹브라우저가 이해할 수 있도록, 구조화된 정보를 필요로 한다.

HTML을 배워서 웹콘텐츠 브라우저에서 잘 나오도록 표현하는 방법을 배운다.

## 학습 목표

- HTML로 간단한 웹사이트를 만들 수 있다.

## What I learned 

- html 기본 태그의 사용방법 
- 시멘틱한 태그의 작성
- BEM 을 통한 class 명 작성

## What I see

- 모던 웹을 위한 HTML5+CSS3 바이블, 윤인성
- 패스트캠퍼스 프론트엔드 입문 강의
